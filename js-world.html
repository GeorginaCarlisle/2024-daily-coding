    Events

    JavaScript event listeners listen out for changes occuring within the page triggering functionality when these events occur.

    Functionality could include http requests or alterations to the DOM.

    There are many different types of event that JavaScript can listen out for, with a number of these utilised in this page.

    The in-page JavaScript Log displays information about these as they happen.

    These include load events, mouse events, click events and keyboard events. What can you make happen?
